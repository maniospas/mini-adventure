# Play in your browser

itch.io: https://maniospas.itch.io/mini-adventure

github (latest features, a bit unstable): https://maniospas.github.io/mini-adventure/

## Dev or

Clone this repo, edit *index.html* (if you want to make changes to the game) and serve it with the following small server to be able to load its image data:

```bash
python3 -m http.server
```
